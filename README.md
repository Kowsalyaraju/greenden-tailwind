# greenden-tailwind
🌿 Greenden – Plant Store Website (Tailwind CSS Project)

Greenden is a responsive multi-page plant store website built using Tailwind CSS. This project demonstrates modern UI design, responsive layouts, and smooth navigation between pages.

The website consists of three main pages: Home, Products, and Contact. The Home page provides an overview of the store and includes navigation links that seamlessly direct users to the Products and Contact pages. The Products page showcases a variety of plants with images, names, and pricing, while the Contact page provides essential details for user communication.

The project features a consistent navbar and footer across all pages, ensuring a uniform user experience. Tailwind CSS utility classes are used extensively to create a clean, responsive, and visually appealing design.

Key Features:

🌱 Multi-page website (Home, Products, Contact)
🔗 Smooth navigation between pages
🪴 Product listing with images and details
📱 Fully responsive design using Tailwind CSS
🧭 Reusable components (Navbar & Footer)
🎨 Modern and minimal UI design

This project highlights my ability to build structured layouts, implement responsive design principles, and create user-friendly interfaces using Tailwind CSS.
